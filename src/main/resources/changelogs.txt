Version Release v1.2.2 (Fabric/Quilt: 1.21(.1)

  Additions
•   Added compatibility for the latest version of Excessive Building

  Changes
•   Updated the tooltip for all wool variants from "A soft layer dyed..." to "A soft cover dyed..."
•   Updated the tooltip for all grate blocks from "[block name] that has been chiseled, can be waterlogged" to "[block name] that has had gaps chiseled into it"
•   Updated the tooltip for normal candles from "A light source that can create a bigger light the more there are on one space"
...to "A source of light that gets bigger when more are placed in one space"
•   Updated zh_tw (@yichifauzi)
•   Updated contributor capes

Version Release 1.2.0 (Fabric/Quilt: 1.21)

  Additions
•   Added tooltips for flow and bolt armor trims

  Changes
•   Tooltips will no longer display if they are empty (great for modpack creators that do not want specific tooltips in game)
•   The mod now requires Forge Config API Port instead of Cloth Config
•   The mod now requires at least Fabric Loader v0.16
•   Shift to display tooltips is now on by default

  Fixes
•   Contributor capes are no longer overridden by vanilla capes
